# NeuroStartUp
Динамически развивающийся стартап, специализирующийся на поиске с использованием новейших технологий искусственного интеллекта.
## Начало работы
Для начала работы необходимо:
1. Скачать на свой ПК дистрибутив проекта с официального сайта;
1. Установить необходимое ПО;
1. Выполнить необходимые настройки в соответствии с инструкцией;
### Prerequisites
* Операционная система: Windows 8-10, MacOS.
* Архиватор (например Winrar).
* Браузер, обновлённый до последней версии (например Google Chrome).
### Установка и запуск
1. Распаковать скачанный архив в папку проекта.
1. Запустить установку, кликнув на ярлык install.exe
1. Выполнять указанные действия.
1. После установки на Рабочем столе кликнуть на появившийся ярлык проекта.
1. Настроить проект в соответствии с инструкцией.
## Лицензия
После устаноки и первого запуска проекта начнётся пробный бесплатный период длительностью 2 недели.
Затем Вы должны будете приобрести лицензию на данный продукт.
